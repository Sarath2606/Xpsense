# XPenses

A modern, mobile-first expense tracker built with React + Tailwind CSS.

## Run locally

```bash
npm install
npm start
```

If port 3000 is busy, accept the prompt to use another port.

## Build

```bash
npm run build
```

## Project structure

- `src/` React source code
- `public/` static assets and HTML

## Tech
- React (CRA)
- Tailwind CSS
